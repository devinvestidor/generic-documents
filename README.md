# generic-documents
Este projeto foi desenvolvido no curso Java Web - Senai. O mesmo teve foco principal para as classes genéricas de:
* Entity
* DAO
* Managed Bean

## Tecnologias Utilizadas
* XHTML
* CSS
* JavaScript
* Java
* JSF
* JPA + Hibernate
* PrimeFaces 
* Tomcat
